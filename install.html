Установщик Модификации Altbeta для Betacraft

Эта страница позволит вам скачать, или обновить уже существующий установщик.

Инструкция по установке

    1. Скачайте Java 8 с сайта https://www.azul.com/
    2. Скачайте лаунчер betacraft с сайта https://betacraft.uk
    3. Скачайте и запустите Altbeta_installer, скачав его ниже
    ФайлMD5
    Altbeta_installer.py1E77DFF4EB6FC942F1484F1FC66E79FD
    Altbeta_installer.exe752C3842F9A299877C42F4F3A7F3B2D7
    Проверьте что всё установилось, в программе должен быть примерно следующий вывод:
Available versions:
2: 300524.zip (Released on 2024-05-30)
Enter version number to install (default is latest, 2):
[==================================================] 100%
Altbeta has been successfully installed into Betacraft Launcher.
Press enter to exit the program.
    4. Запустите Betacraft и найдите версию "altbeta"
    5. Всё должно запустится!
